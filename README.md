# book-manager

> PDF图书管理系统，管理本地的PDF图书，进行分类，添加描述信息等操作

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

```
#### 使用技术
<li>框架：Electron-vue</li>
<li>界面：Element-ui</li>
<li>数据库：IndexedDB</li>
<li>生成PDF的界面预览：Pdfjs</li>
